# InsightForge Web (Next.js)

## Getting Started

```bash
cd client
npm install
npm run dev
```

The app expects `NEXT_PUBLIC_API_URL` to point to the FastAPI backend (default `http://localhost:8000/api`). Create a `.env.local` file if you need to override it.

## Available Scripts

- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run start` – run the production build locally
- `npm run lint` – run ESLint checks
- `npm run format` – format the project with Prettier

## Tech Stack

- Next.js App Router
- TypeScript & Tailwind CSS
- Axios for API communication
